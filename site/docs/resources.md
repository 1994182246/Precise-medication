# 常用数据库资源

以下是精准治疗和药物基因组学领域常用的数据库和资源链接：

## 综合药物基因组学数据库

*   **PharmGKB (The Pharmacogenomics Knowledgebase)**
    *   链接: [https://www.pharmgkb.org/](https://www.pharmgkb.org/)
    *   简介: 权威的药物基因组学知识库，提供临床指南、药物标签注释、基因-药物关系等信息。

*   **CPIC (Clinical Pharmacogenetics Implementation Consortium)**
    *   链接: [https://cpicpgx.org/](https://cpicpgx.org/)
    *   简介: 临床药物遗传学实施联盟，提供基于基因检测结果的药物剂量调整指南。

*   **FDA Table of Pharmacogenomic Biomarkers in Drug Labeling**
    *   链接: [https://www.fda.gov/drugs/science-and-research-drugs/table-pharmacogenomic-biomarkers-drug-labeling](https://www.fda.gov/drugs/science-and-research-drugs/table-pharmacogenomic-biomarkers-drug-labeling)
    *   简介: FDA 发布的药物标签中的药物基因组学生物标志物列表。

*   **DPWG (Dutch Pharmacogenetics Working Group)**
    *   链接: [https://www.knmp.nl/patientenzorg/medicatiebewaking/farmacogenetica](https://www.knmp.nl/patientenzorg/medicatiebewaking/farmacogenetica)
    *   简介: 荷兰药物遗传学工作组，提供欧洲的药物基因组学指南。

## 基因与变异数据库

*   **dbSNP (Single Nucleotide Polymorphism Database)**
    *   链接: [https://www.ncbi.nlm.nih.gov/snp/](https://www.ncbi.nlm.nih.gov/snp/)
    *   简介: NCBI 提供的单核苷酸多态性数据库。

*   **Allele Frequency Net Database (AFND)**
    *   链接: [http://www.allelefrequencies.net/](http://www.allelefrequencies.net/)
    *   简介: 提供全球不同人群中免疫相关基因（如HLA）的等位基因频率数据。

*   **1000 Genomes Project**
    *   链接: [https://www.internationalgenome.org/](https://www.internationalgenome.org/)
    *   简介: 人类遗传变异的详细目录。

*   **NCBI Gene**
    *   链接: [https://www.ncbi.nlm.nih.gov/gene/](https://www.ncbi.nlm.nih.gov/gene/)
    *   简介: 提供基因的详细信息，包括序列、表达、功能等。
