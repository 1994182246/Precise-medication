# 基础知识

## 一、药理基因组学概述

### 1.1 什么是药理基因组学

药理基因组学是研究基因变异如何影响药物疗效和安全性的交叉学科。它结合了药理学、遗传学和基因组学的方法，旨在实现个性化药物治疗。

与传统药理学不同，药理基因组学关注个体差异，特别是遗传因素导致的药物反应变异。

发展历程：
- 20世纪中叶：发现药物代谢酶遗传多态性
- 1990s：人类基因组计划启动
- 2000s：药理基因组学概念提出
- 近年来：临床应用普及

### 1.2 精准医疗的概念

精准医疗（Precision Medicine）是指根据个体基因、环境和生活方式特征，提供针对性的医疗干预。

药理基因组学是精准医疗的核心组成部分，通过基因检测指导药物选择和剂量调整。

相关概念：
- 个性化用药（Personalized Medicine）
- 基因组学（Genomics）
- 转化医学（Translational Medicine）

### 1.3 药理基因组学的意义

- **提高治疗效果**：选择最适合的药物和剂量
- **减少不良反应**：避免严重副作用
- **降低医疗成本**：减少试错治疗
- **推动创新**：加速新药研发

## 二、人类基因组与药物代谢基础

### 2.1 人类基因组简介

人类基因组包含约30亿个碱基对，编码约2万个基因。

**基因变异类型**：
- 单核苷酸多态性（SNP）：最常见的变异
- 插入/缺失（Indels）
- 拷贝数变异（CNV）
- 结构变异

SNP在药物基因组学中特别重要，因为它们可能影响酶活性。

### 2.2 药物代谢酶系统

#### 细胞色素P450（CYP）酶族
CYP酶负责大多数药物的氧化代谢。

**主要CYP酶**：
- **CYP2D6**：代谢可待因、氟西汀等
- **CYP2C9**：代谢华法林、非甾体抗炎药
- **CYP2C19**：代谢奥美拉唑、氯吡格雷
- **CYP3A4/5**：代谢多数药物

#### 其他重要酶
- **UGT酶**：葡萄糖醛酸化反应
- **NAT酶**：乙酰化反应
- **TPMT酶**：硫嘌呤代谢

### 2.3 药物转运蛋白

药物转运蛋白影响药物的吸收、分布和排泄。

**主要转运蛋白**：
- **P-糖蛋白（P-gp）**：ABC转运蛋白 superfamily
- **BCRP**：乳腺癌耐药蛋白
- **MRP2**：多药耐药相关蛋白

基因变异可能导致药物积聚或排泄过快。

## 三、基因变异与药物反应

### 3.1 药代动力学变异

药代动力学（Pharmacokinetics）研究药物在体内的ADME过程。

**酶活性分型**：
- **快速代谢者（EM）**：酶活性正常
- **慢代谢者（PM）**：酶活性降低
- **超快代谢者（UM）**：酶活性增强
- **中间代谢者（IM）**

**案例**：CYP2D6慢代谢者对可待因的镇痛效果减弱。

### 3.2 药效动力学变异

药效动力学（Pharmacodynamics）研究药物与靶点的相互作用。

**靶点变异**：
- 受体基因多态性
- 信号通路基因变异

**案例**：VKORC1基因变异影响华法林的抗凝效果。

### 3.3 药物不良反应与基因

遗传因素可导致：
- **过敏反应**：如HLA-B*57:01与阿巴卡韦
- **剂量相关毒性**：如TPMT与硫嘌呤类药物
- **特异性不良反应**

## 四、基因检测技术

### 4.1 常用检测方法

- **PCR-based方法**：实时荧光定量PCR
- **基因芯片技术**：高通量SNP检测
- **下一代测序（NGS）**：全基因组或靶向测序

### 4.2 检测流程

1. 样本采集（血液、口腔拭子）
2. DNA提取
3. 基因分型
4. 数据分析
5. 报告生成

### 4.3 检测的局限性

- 检测范围有限
- 成本较高
- 某些变异难以检测
- 结果解读复杂

## 五、临床应用指南

### 5.1 基因检测时机

- 新治疗开始前
- 药物无效或不良反应时
- 高风险药物使用前
- 家族史提示遗传风险时

### 5.2 结果解读

- 基因型 → 表型预测
- 临床证据等级（A-D级）
- 置信区间和不确定性

### 5.3 用药调整策略

- **剂量调整**：基于代谢酶活性
- **药物替代**：选择不依赖该酶的药物
- **监测加强**：增加血药浓度监测
- **预防措施**：避免高风险组合
