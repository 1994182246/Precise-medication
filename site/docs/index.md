# 精准治疗药物知识库

欢迎来到精准治疗药物知识库。本网站旨在提供中国上市的精准治疗药物的药物基因组学知识，包括药物简介、相关基因、基因对疗效/不良反应的影响以及临床用药指导。

## 主要内容

- **抗凝和抗血小板药物**: 华法林, 氯吡格雷
- **抗癫痫药**: 卡马西平, 苯妥英
- **抗抑郁药**: 氟西汀, 氟伏沙明
- **质子泵抑制剂**: 奥美拉唑
- **降脂药**: 阿托伐他汀
- **抗肿瘤药**: 氟尿嘧啶
- **抗结核药**: 异烟肼
- **抗真菌药**: 伏立康唑
